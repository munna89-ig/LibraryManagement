# Library Management System

## Overview

A desktop-based Role-Based Library Management System developed using C# .NET (Windows Forms).

The system automates and manages core library operations including book management, borrowing, reserving, fine calculation, fine payment, and report generation.

It supports three different user roles:

- Admin  
- Librarian  
- Member  

Each role has secure authentication and controlled access to specific system functionalities.

---

## Authentication & Access Control

- User Registration (Signup)
- Secure Login System
- Forgot Password / Password Recovery
- Role-Based Access Control
- Separate Dashboards for Admin, Librarian, and Member

---

## Admin Panel

- Manage Librarians
- Manage Members
- Monitor System Activities
- Access System Reports
- Control Overall System Settings

---

## Librarian Panel

- Add / Update / Delete Books
- Issue Books to Members
- Manage Return System
- Handle Book Reservations
- Track Borrow History
- Generate Reports

---

## Member Panel

- View Available Books
- Search Books
- Borrow Books
- Reserve Books
- View Borrow History
- Check Fine Status
- Pay Fines

---

## Fine & Payment System

- Automatic Fine Calculation for Late Returns
- Fine Payment Management
- Fine Status Tracking
- Fine Reports

---

## Reports

- Borrow Report
- Reservation Report
- Fine Report
- User Activity Report

---

## Technologies Used

- C#
- .NET Framework
- Windows Forms
- SQL Database


---

## Project Objective

The objective of this project is to digitize traditional library management processes by creating a secure, efficient, and user-friendly desktop application with role-based access control.
